# GigHub API

A FastAPI-based REST API for managing freelance gig listings in Nairobi.

## Features
- List all gigs (with optional filtering by category and budget)
- View a single gig by ID
- Search gigs by title
- Create a new gig
- Update a gig's budget or status
- Delete a gig

## Technologies
- FastAPI
- Python 3.x
- Uvicorn (ASGI server)
